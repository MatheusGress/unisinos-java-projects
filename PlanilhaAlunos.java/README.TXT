# 🎓 Sistema de Alunos — Exercício em Java

## 📖 Descrição

Este projeto foi desenvolvido em **Java** como exercício prático para aplicação dos conceitos de **Programação Orientada a Objetos (POO)**. O sistema realiza o cadastro de alunos, cálculo de média final e manipulação de informações através de métodos e interação via teclado.

---

## 🧱 Estrutura do Projeto

O projeto possui duas classes principais:

### 👨‍🎓 Classe `Aluno`

Responsável por representar um aluno e armazenar suas informações acadêmicas.

**Atributos:**
- nome
- matrícula
- nota do Grau A
- nota do Grau B

**Funcionalidades:**
- Construtor para inicialização dos atributos
- Métodos de acesso (getters e setters)
- `calculaMediaFinal()` — calcula a média final utilizando a fórmula:


- `imprimeInfo()` — imprime todas as informações do aluno de forma organizada, incluindo sua média final

---

### ▶️ Classe `PrincipalAluno`

Classe responsável pela execução do programa, contendo o método `main`.

Durante a execução, o programa:

- Cria três alunos com dados previamente definidos
- Imprime as informações completas dos alunos
- Altera valores específicos de atributos
- Exibe informações individuais conforme solicitado
- Solicita via teclado um novo valor para a nota do Grau B de um aluno

---

## 🧠 Conceitos Aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Construtores
- Manipulação de objetos
- Métodos de cálculo
- Entrada de dados pelo teclado
- Organização de saída no console

---

## 🚀 Como Executar

1. Abra o projeto no **BlueJ** (ou outra IDE Java).
2. Compile as classes do projeto.
3. Execute a classe `PrincipalAluno`.
4. Informe o valor solicitado quando o programa pedir entrada pelo teclado.

---

Praticar a criação e manipulação de classes em Java, reforçando conceitos fundamentais de orientação a objetos e lógica de programação.
