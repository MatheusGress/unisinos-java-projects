# 📚 Projetos Acadêmicos — Java | Unisinos

Este repositório reúne pequenos projetos desenvolvidos durante minha graduação na **Unisinos**, com o objetivo de registrar minha evolução prática na área de desenvolvimento de software.

Os projetos foram construídos inicialmente utilizando **Java** no ambiente **BlueJ**, conforme a proposta das disciplinas introdutórias de Programação Orientada a Objetos.

A ideia deste repositório é documentar minha evolução técnica ao longo da faculdade, aplicando conceitos fundamentais de programação, organização de código e versionamento com Git.

---

## 🎯 Objetivo

- Consolidar conceitos aprendidos em aula
- Praticar Programação Orientada a Objetos (POO)
- Manter um histórico público de evolução acadêmica
- Aplicar boas práticas de versionamento com Git e GitHub
